> Send any data via a QR code without having to log in / install anything

![screenshot](ui.png)

## Usage

- 🌐 Open [QRS](https://qrs.snlx.net) on the first device
- 📱 Scan the QR code with the second device
- 📨 Click one of the buttons

## Supported Data

- [x] 🖹 Text
- [x] 🔗 Links
- [x] 📂 Files
- [x] 🖼️ Images
- [ ] 🎤 Audio
